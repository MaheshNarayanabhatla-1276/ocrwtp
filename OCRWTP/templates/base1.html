OPTICAL CHARACTER RECOGNITION WITH TEXT PROCESSING

OCRWTP is a web application which extract text information from image which is printed one and converts image text to summarized data. It has an additional feature to translate the summarized text.

Watch tutorial

FEATURES

UPLOAD

Upload an image which is printed form by scanning and saving it on the desktop. make sure the image is only in printed format, no written image is accepted. It supports JPG, JPEG, PNG and BMP files.

SUMMARY

The extracted text from the uploaded image is given otput as a summarized text in few linesof text, which is easy to understand and interpret.

TRANSLATION
